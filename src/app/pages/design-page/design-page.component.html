Grid

Todo o layout da intranet deve ser baseado em um grid. Em telas com uma coluna, onde normalmente não usamos uma sidebar, o conteúdo tem a largura total do espaço, é necessário inserir uma classe para que o grid funcione perfeitamente.

O grid sempre deve ter uma estrutura de hierarquia correta de elementos. Um elemento com a classe .row deve envolver os elementos que dividirão a estrutura, que terão classes como .col-1, .col-3, .col-4 até .col-12. Abaixo segue um exemplo com as grids e suas respectivas classes.
